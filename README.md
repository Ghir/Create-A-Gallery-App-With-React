## React Photo Gallery

Insert a search keyword and view the photos fetched from Flickr.

### Config file

Add a `config.js` file to `src` with your API key:

```
const apiKey = "YOUR-API-KEY";
export default apiKey;
```

### Running

- `npm install`
- `npm start`
